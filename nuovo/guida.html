Guida Completa alla Grid Bootstrap 5

Impara a usare il sistema a griglia di Bootstrap per layout responsivi.

1. Introduzione alla Grid

Il sistema a griglia di Bootstrap utilizza una serie di contenitori, righe e colonne per allineare e disporre i contenuti. È costruito con flexbox ed è completamente responsive. Si basa su un sistema a 12 colonne.

Concetti chiave:

  • .container o .container-fluid: Definiscono la larghezza e l'allineamento del contenuto.
  • .row: Contenitori per le colonne. Eliminano il padding del container e applicano margini negativi per compensare il padding delle colonne.
  • .col, .col-*, .col-sm-*, ecc.: Le colonne effettive dove inserire il contenuto. Hanno un padding orizzontale (chiamato "gutter").

2. Visualizzare i Bordi delle Colonne

Per capire meglio come sono posizionate le colonne, è utile aggiungere dei bordi. Possiamo farlo con un po' di CSS. In questa guida, tutte le colonne negli esempi avranno automaticamente i bordi grazie alla classe .visualize-cols applicata al loro contenitore .row o .example-container.

Il CSS utilizzato in questa pagina per visualizzare i bordi è:


.visualize-cols [class^="col"] {
    border: 1px solid #007bff;
    background-color: rgba(0, 123, 255, 0.05);
    padding-top: .75rem;
    padding-bottom: .75rem;
    margin-bottom: 1rem;
    text-align: center;
}

Basterà avvolgere le tue righe (.row) in un div con classe .visualize-cols o applicare stili simili direttamente alle classi .col.

3. Contenitori: .container e .container-fluid

I contenitori sono l'elemento più esterno necessario per la griglia.

  • .container: Fornisce un contenitore a larghezza fissa responsive (la sua max-width cambia ad ogni breakpoint).
  • .container-fluid: Fornisce un contenitore a larghezza piena, che occupa tutta la larghezza del viewport.

4. Righe e Colonne di Base

Le righe (.row) creano gruppi orizzontali di colonne. Le colonne (.col) specificano quante delle 12 colonne disponibili occuperanno.

Colonne di Larghezza Automatica (.col)

Se usi solo la classe .col senza specificare un numero, Bootstrap distribuirà lo spazio equamente tra le colonne presenti nella riga.

5. Breakpoint della Grid

Bootstrap ha 6 breakpoint di default (tier), che permettono di controllare il layout a diverse dimensioni di schermo:

  • xs (extra small): <576px (implicito, non serve il prefisso xs, es. .col-6)
  • sm (small): ≥576px (es. .col-sm-6)
  • md (medium): ≥768px (es. .col-md-6)
  • lg (large): ≥992px (es. .col-lg-6)
  • xl (extra large): ≥1200px (es. .col-xl-6)
  • xxl (extra extra large): ≥1400px (es. .col-xxl-6)

Puoi combinare queste classi per creare layout molto flessibili. Le classi si applicano dal breakpoint specificato in su, a meno che non vengano sovrascritte da un breakpoint più grande.

6. Gutters (Spaziatura tra Colonne)

I "gutters" sono gli spazi tra il contenuto delle colonne, creati dal padding orizzontale. Le righe (.row) hanno margini negativi per compensare il padding della prima e ultima colonna.

Puoi controllare i gutters usando le classi .g-*, .gx-* (orizzontale), .gy-* (verticale) sulla classe .row. I valori vanno da 0 a 5.

7. Offsetting delle Colonne

Puoi spostare le colonne a destra usando le classi .offset-*. Ad esempio, .offset-md-2 sposta una colonna di due unità a partire dal breakpoint md.

Codice Esempio:


  

    
.col-md-4

    
.col-md-4 .offset-md-4

  

  

    
.col-md-3 .offset-md-3

    
.col-md-3 .offset-md-3

  

  

    
.col-md-6 .offset-md-3

Risultato:

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
.col-md-3 .offset-md-3
.col-md-6 .offset-md-3

9. Annidamento (Nesting)

Puoi annidare le righe e le colonne. Inserisci una nuova .row e un set di .col-* all'interno di una colonna esistente.

Codice Esempio:


  

    

      Livello 1: .col-sm-9
      

        
Livello 2: .col-8 .col-sm-6

        
Livello 2: .col-4 .col-sm-6
